# Vulnerability Scanner
![Status](https://img.shields.io/badge/Status-Maintenance-yellow)

## Overview
An automated vulnerability assessment tool that performs deep scans for common web application vulnerabilities such as Cross-Site Scripting (XSS) and SQL Injection (SQLi). It utilizes Selenium for dynamic scanning and Owasp ZAP for comprehensive security analysis, generating automated PDF reports.

## Features
- **Automated Scanning**: Scheduled security assessments of target URLs.
- **Vulnerability Checks**: XSS, SQLi, CSRF, and more.
- **Dynamic Analysis**: Selenium-driven interaction with web elements.
- **Automated Reporting**: Detailed PDF reports for security teams.

## Technical Stack
- **Language**: Python
- **Tools**: Selenium, Owasp ZAP
- **Category**: Automation / PenTesting
