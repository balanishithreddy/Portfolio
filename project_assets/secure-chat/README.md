## Overview
SecureChat (rebranded as MailShield) is an end-to-end encrypted messaging platform. It ensures complete privacy for communications using modern cryptographic protocols. Features include self-destructing messages, multi-factor authentication, and advanced server-side shielding to prevent data breaches.

## Features
- **End-to-End Encryption**: Zero-knowledge privacy for all messages.
- **Self-Destructing Messages**: Automatic deletion of sensitive communications.
- **Multi-Factor Authentication (MFA)**: Enhanced identity verification.
- **Advanced Shielding**: Server-side protection against unauthorized data access.

## Technical Stack
- **Frontend**: React
- **Backend**: Node.js
- **Real-time**: Socket.io
